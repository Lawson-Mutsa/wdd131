Site Plan - Personal Portfolio

Site Name:

LawsonMutsa.com

This name represents a portfolio site that showcases my web development projects, skills, and ways to contact me.

Optional Domain Availability: lawsonmutsa.com

Site Purpose:

The site serves as a professional portfolio to showcase my web development skills and projects. It includes an introduction to myself, a gallery of projects I've worked on, and a way for potential employers or collaborators to contact me.

Scenarios:

  • What web development skills do you specialize in?
  • How can I contact you for collaboration or job opportunities?

Color Schema:

Primary Color: Dark Blue (#2c3e50) - Used for headings and important sections.

Secondary Color: Light Gray (#f4f4f9) - Used as the background color for a clean, modern look.

Accent Color: Red (#e74c3c) - Used for highlighting key sections or call-to-action buttons.

Typography:

Font: 'Open Sans' - This font will be used for all body text and general content.

Headings: 'Open Sans' - Large and bold for easy readability on both desktop and mobile devices.

Wireframe:

Here is a simple layout for the homepage:

  • Mobile View: A single-column layout with the header at the top, followed by an introductory section, portfolio section, and contact form at the bottom.
  • Wide Desktop View: A multi-column layout where the portfolio section is displayed in a grid, the introduction is on the left, and the contact form is on the right.
